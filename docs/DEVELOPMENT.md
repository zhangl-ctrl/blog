# 开发记录

## 2026-03-11

### 1. 项目初始化

- 使用 Vite 创建 Vue 3 项目
- 安装依赖：vue, vue-router, marked, @vitejs/plugin-vue, vite

### 2. 基础页面开发

- 创建首页 (`Home.vue`) - 展示文章列表
- 创建文章详情页 (`Post.vue`) - 渲染 Markdown 文章，支持评论
- 创建关于页 (`About.vue`) - 个人介绍
- 配置路由

### 3. UI 优化

- 添加 Google Fonts（Noto Serif SC + Cormorant Garamond）
- 实现毛玻璃效果导航栏
- 添加页面过渡动画
- 美化文章卡片样式

### 4. 文章编辑器功能

- 创建 `/editor` 路由
- 实现左侧编辑 + 右侧预览布局
- 支持 Markdown 实时预览
- 添加"保存草稿"功能（localStorage）
- 添加"导出 MD"功能（下载文件）
- 添加"发布文章"功能（保存到 localStorage 并显示在首页）

### 5. Toast 提示组件

- 创建通用提示组件 (`Toast.vue`)
- 支持 success / error / warning / info 四种类型
- 毛玻璃效果，顶部居中显示
- 3秒自动消失

### 6. 文章标签功能

- 编辑器支持添加多个标签（回车或逗号添加）
- 首页显示文章标签
- 首页支持标签筛选（点击标签筛选）
- 首页支持搜索文章（标题、摘要、标签）
- 文章详情页显示标签

### 7. 文章编辑功能

- 文章详情页添加"编辑文章"按钮（仅本地文章显示）
- 编辑器支持加载已有文章数据
- 编辑器 URL 参数传递文章 ID (`/editor?id=xxx`)
- 更新文章时保留原文章 ID 和日期
- 支持修改标题、内容、标签

### 8. 预设标签功能

- 添加 24 个程序员相关预设标签
- 预设标签：前端、后端、JavaScript、TypeScript、Vue、React、Node.js、Python、Go、Java、算法、数据结构、数据库、Redis、Git、Docker、Linux、设计模式、性能优化、面试、职场、个人成长、读书笔记、生活
- 点击预设标签即可选择/取消
- 支持自定义标签输入（回车或逗号添加）
- 已选中的标签会高亮显示
